## Setup

### Installation
#### How to install this project:
    - Clone this repo, run:
        git clone https://github.com/John-Kimani/cli_POS_python.git
    - Move into project folder:
        cd cli_POS_python
    - Open project with IDE i.e VsCode run:
        code .
    - Run entry file:
        python3 main.py
